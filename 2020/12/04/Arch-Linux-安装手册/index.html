Arch Linux 安装手册

2020-12-04

概述

Arch Linux 是很多极客所热衷的 Linux 发行版。其特点是：简洁。

本日志参考 Arch 官方文档，整理了快速安装一个 Arch 环境的步骤。

下载镜像

可以到 Arch 官网 下载最新镜像。

开始安装

首先保证有网。

通过 CD/USB 引导进入镜像的 Linux 环境。

分区

使用 fdisk 分区，对于 MBR BIOS，至少保证以下两个分区：

  • root_partition: /
  • swap_partition: swap

对于 GPT UEFI，应该增加 EFI 分区。

可依据具体情况增设 /home 等分区。

接下来用 BIOS 引导，不考虑 EFI。

格式化

使用 mkfs 格式化根目录为 ext4 格式。

# mkfs.ext4 /dev/root_partition

格式化交换分区：

# mkswap /dev/swap_partition

挂载

挂载 root 分区到安装环境的 /mnt。

# mount /dev/root_partition /mnt

然后可挂载交换分区。

# swapon /dev/swap_partition

安装基础包

安装基础包到 /mnt。

# pacstrap /mnt base linux linux-firmware

配置

到此，安装结束。对系统进行简单配置。

Fstab

# genfstab -U /mnt >> /mnt/etc/fstab

Chroot

# arch-chroot /mnt

Timezone

# ln -sf /usr/share/zoneinfo/Region/City /etc/localtime

生成 /etc/adjtime。

# hwclock –systohc

Localization

# locale-gen

创建 /etc/local.conf。

# echo ‘LANG=en_US.UTF-8’ > /etc/locale.conf

Network

# echo ‘myhost’ > /etc/hostname

然后：

Root Passwd

# passwd

GRUB 引导

安装

安装 GRUB。

# pacman -Sy grub

配置

# grub-install /dev/device_name

然后：

# grub-mkconfig -o /boot/grub/grub.cfg

大功告成

弹出安装介质，重启机器。
